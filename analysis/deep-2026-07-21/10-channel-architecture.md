# Архитектура каналов и тегов

## Зачем столько каналов

Разделение — не техническая случайность, а **психологическая компартментализация** с разными контрактами: с аудиторией и с собой.

```
                    ПУБЛИЧНОЕ                    ПРИВАТНОЕ
              ┌─────────────────────┬─────────────────────┐
  СОМАТИКА    │  Да и Да            │  (личные заметки)   │
              ├─────────────────────┼─────────────────────┤
  МАГИЯ       │  —                  │  omnes colores      │
              ├─────────────────────┼─────────────────────┤
  ИНТЕЛЛЕКТ   │  (будущее)          │  research           │
              ├─────────────────────┼─────────────────────┤
  КУЛЬТУРА    │  —                  │  cultura nova       │
              ├─────────────────────┼─────────────────────┤
  СМЕШАННОЕ   │  VK · Instagram     │  —                  │
              └─────────────────────┴─────────────────────┘
```

| Канал | Платформа | Статус | channel_tag | min_chars | wave_gap |

| **omnes colores** | Telegram | draft | Магический | 40 | 45 мин |

### Да и Да
- **Голос:** публичный терапевтический, тёплый, проводящий
- **Ритм:** короткие волны — живой поток ощущений
- **Аудитория:** клиенты, читатели соматики, «свои»
- **Риск:** слияние с личным дневником без границы

### omnes colores («все цвета»)
- **Голос:** Лун, магический, без обязательства быть полезным
- **Функция:** право на необъяснимое, сны, таро, символ
- **Принцип:** оставить **draft** — не всё должно стать хроникой

### исследование и есть терапия
- **Голос:** интеллект + терапия; тезис в названии
- **Формат:** длиннее, с комментариями — фрагмент исследования, не пост
- **Потенциал:** флагман глубины при публикации

### Cultura nova
- **Голос:** культура, практика мышления
- **Особенность:** чат «Рефлексия» — meta-уровень
- **Связь:** пересечение с городом, языком, искусством

### VK
- **Голос:** смешанный, интимно-публичный
- **Содержание:** от «зацвели липы» до длинных кризисных нарративов и анонсов продуктов
- **Особенность:** лайки в corpus → meta-блоки (скрыты на сайте)

### Instagram
- **Голос:** визуальный + длинная подпись
- **Фильтр corpus:** подписи ≥80 символов, нужны фото
- **Темы:** тело, эстетика, город

---

## Почему splits работают

1. **Публичное vs интимное** — разная цензура и глубина
2. **Регистр языка** — соматика ≠ магия ≠ исследование
3. **Стадия готовности** — draft = созревание до хроники
4. **Гранулярность** — research требует 120+ символов и тредов

## Тематические теги (theme_tags)

| Тег | Regex-смысл | Свет / тень |
|-----|-------------|-------------|
| **тело** | телесность, дыхание, таз, фасции | Свет — ось публичной идентичности |
| **город** | улицы, Екатеринбург, дворы, трамваи | Свет — мета-тема всей хроники |
| **практика** | йога, медитация, ритуал | Свет — Да и Да |
| **язык** | текст, стих, музыка, голос | Мост между всеми голосами |
| **горе** | утрата, слёзы, прощание | Тень — редко на лендинге |
| **путь** | странствие, переезд, паломничество | Связь с эпохами-переездами |
| **код** | git, symfony, деплой | Профессиональный Артур |
| **сновидение/таро** | магия, архетипы, сны | Тень — omnes colores |
| **зеркало** | рефлексия, ОС | Мета-тег проекта + analysis |

### Созвездие

```
         [язык]
            │
    [практика]──[тело]──[город]
            │      │
         [путь]  [зеркало]
            │
    ┌───────┴───────┐
 [код]           [горе]
                    │
            [сновидение/таро]
```

**Центр:** тело – город – практика  
**Орбита:** язык, зеркало  
**Периферия/тень:** горе, сновидение/таро  
**Тангент:** код (дневная работа) vs путь (ночная мобильность)

## Рекомендации по архитектуре

1. **Не сливать** omnes и Да и Да — разная функция психики
2. **План публикации research** — 10–20 текстов, не весь корпус сразу
3. **VK** — оставить «смешанным»; не форсировать один тег
4. **Хроника на сайте** — фильтры по каналу/эпохе/тегу уже заложены в продукт
5. **analysis/** — зеркала эпох дополняют тег «зеркало» мета-уровнем
