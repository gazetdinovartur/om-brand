# Зеркала эпох

Биография как география: каждый период жизни назван **местом** — улицей, районом, регионом или проектом. Ниже — психологическое зеркало по `config/content/catalog.json`.

---

## Зачем эта система

Адрес как имя эпохи фиксирует **тело в пространстве**. Переезд — смена режима бытия, не фон. Для соматического мышления это метод, не метафора.

**Эволюция именования:**
1. Прямые топонимы (Щорса, Добролюбова)
2. Мифы (Рябиновое государство, Народная воля)
3. Регионы (Татарстан, Зеленоград)
4. Проекты вне календаря (Делайогу)

## Паттерны времени

### Пробелы
- **12.2015 – 02.2016** (~4 мес.) — между Щорса и Добролюбова; возможный кризис без адреса для имени

### Наложения
- **Осень 2016:** Добролюбова + Рябиновое государство + Ленина пять — тройное пространство
- **Лето 2020:** Народная воля ∩ Странствия — воля в движении
- **2024:** Зеленоград ⊂ Рассветная

### Возвраты
- **Ленина пять** — единственная discontinuous era: 2016–2018, пауза ~7 лет, **возобновление 08.2025** после Рассветной
- **Коммуна** — return inside return: новый социальный режим на старом адресе

### Ритм
- **2012–2017:** частая смена (6 эпох / 5 лет)
- **2018–2021:** ~год на адрес
- **2021–2025:** Рассветная как длинная платформа (~4 года)
- **2025–2026:** возврат + вложенная Коммуна

---

## era_priority

При конфликте дат побеждают **короткие специальные эпизоды** (коммуна, ботаника, зеленоград) над длинным фоном (рассветная, ленина пять). Логика: *исключение важнее правила*.

---

## Как использовать зеркала

1. При чтении старого текста — спросить: *каким «я» это написано?*
2. При писании сейчас — отметить эпоху осознанно (Ленина пять + Коммуна?)
3. В терапии — адрес как якорь: «где в теле этот двор?»

Подробнее по каждой эпохе — в `eras/*.md`.
