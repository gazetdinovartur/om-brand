# Методология и источники

## Задача анализа

Понять через язык и структуру контента:

- кто говорит и из каких «регистров»;
- как устроена внутренняя карта (тело, город, код, магия, горе);
- что сильно, что уязвимо, что остаётся в тени;
- куда может развиваться человек и профессионал;
- какие формы терапии и самопомощи резонируют с этим материалом.

### Полный прогон (локально, где есть `content/`)

```bash
python3 scripts/corpus_build.py --instagram
python3 scripts/run_analysis_pipeline.py
```

Создаёт:
- `corpus/chronicle_entries.jsonl` + `corpus/manifest.json`
- `analysis/deep-2026-07-21/data/corpus-stats.json`
- `analysis/deep-2026-07-21/mirror/` — by-era, by-channel, manifest.json

Синхронизация на cloud agent (пример):

```bash
rsync -avz corpus/chronicle_entries.jsonl HOST:/path/to/om-brand/corpus/
rsync -avz content/ HOST:/path/to/om-brand/content/   # или только corpus после сборки
```

## Методы

1. **Психолингвистика** — лексические поля, синтаксические паттерны, переключение регистров, ритм и дыхание текста.
2. **Интегральный подход (AQAL-lite)** — четыре квадранта: внутреннее/внешнее × индивидуальное/коллективное.
3. **Нарративный анализ** — эпохи как сюжетные акты, каналы как «голоса», теги как тематические орбиты.
4. **Проективная гипотеза** — слепые пятна формулируются как вопросы, не как диагнозы.
5. **Клиническая осторожность** — рекомендации по терапии как *направления поиска*, не назначения.

## Ограничения

- Нет доступа к приватным каналам (omnes colores, research, culture) — анализ их *архитектуры*, не содержания.
- Instagram и полный VK-корпус не прочитаны построчно — только публичный срез.
- Анализ сделан «снаружи» текста; субъективная проверка автором обязательна: *«резонирует / не резонирует / это не про меня»*.
